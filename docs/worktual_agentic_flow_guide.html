1. Core Principle

Worktual uses a two-layer architecture:

LayerRole
Gemini (LLM)Routing, planning, coding, review, conversation
Python (backend)Identity, tool filtering, validation, staging, QA, commit, rollback

Entry point: User message → backend/api/generation.py → backend/agents/orchestration/runner.py

Source of truth: backend/AGENTIC_EXECUTION_FLOW.md, backend/agents/agent_tool_catalog.py, backend/agents/runtime_agents/registry.py

2. End-to-End Flow

User message → generation API → Intent Router Agent → Intent? (greeting | simple_code | needs_confirmation | website_generation | website_update | …) → [optional] Requirement Confirmation Agent (confirm / revise / cancel) → Execution pipeline: website_generation → Greenfield Generation Engine → Streaming File Agent / Parallel Workers website_update → ScopeEngine + Streaming File Agent simple_code → Simple Code Writer Agent → Commit (WRITE_PROJECT_FILES) → Memory Agent (persist summary)

6. Dynamic Agent Registry

Core (7): Intent Analyzer, Requirement Analyst, Task Decomposer, Agent Registry, Workflow Planner, Supervisor, Memory

Reusable specialists (8): Domain Research, UX/Layout, Content, Component/UI, Code Generator, Validation, Preview QA, Repair

Tool policy: Dynamic agents may only use READ_PROJECT_FILES and LOAD_PROJECT_MEMORY — no writes.

When: ENABLE_FULL_DYNAMIC_GENERATION=true

8. Default Production Paths

Website Generation

Intent Router → Confirmation (if enabled) → Greenfield Generation Engine → plan_greenfield_parallel_tasks → Streaming File Agent + blueprint OR Parallel File Workers (rich CRM) → Minimum deliverable check → auto-resume if scaffold-only → Syntax guard → Vite build → Visual QA → WRITE_PROJECT_FILES

Website Update

Intent Router → [optional Confirmation] → Streaming File Agent → ScopeEngine (memory + code index + LLM) → Scope enrichment (handlers, styles, interaction wiring) → str_replace / write_file on scoped files → Commit pipeline + gates

Simple Code

Intent Router → Simple Code Writer Agent → single artifact → validate

Conversation

Intent Router → Conversation Agent → no file writes
