## Quick Reference: What Actually Runs on a Typical Website Build (Default Config)

```
Intent Router Agent          → routing
Supervisor Agent             → loop control
Memory Agent                 → read/load
Prompt Analyst Agent         → brief
Planner Agent                → plan
Code Agent                   → generate + write
Validation Agent             → validate
Preview Agent                → staged preview
Visual QA Agent              → QA
Memory Agent                 → persist
```

**Not run by default:** Agent Registry Agent, UX Review Agent, Accessibility Agent, dynamic specialists (unless `ENABLE_FULL_DYNAMIC_GENERATION=true` or large scoped update).
